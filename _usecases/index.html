Use Cases

To guide agencies through the various requirements outlined in the 800-63-3 suite, NIST has provided several use cases around identity proofing, authentication, and federation.

To provide feedback regarding a current use case, or to suggest your own use case, visit the Contribute page. Use cases should be agency agnostic, simple to understand, and broadly applicable. To assist agencies with use case creation, a Use Case Template has been provided below.

Feedback? Create an issue on the code repository or email us at info@nist.gov.
